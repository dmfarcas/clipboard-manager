Time format

Clipboard Manager stores time as Unix time in a database and formats it using Moment.js. You can create your own string. Please check the documentation.

Change hotkeys

Any combination can be used, pretty much. Except the system wide copy shortcut. Electron's behavior is to make the shortcut exclusive for itself. If you choose CTRL - C, for example, your system's copy shortcut will not work.
